Types of QR Codes

Depending on the type of QR Code, you will be required to fill some fields, if all fields are left empty then no QR Code will be generated

Profile

    The QR Code will point to your profile page

    Please insure that you profile is public

Current Page

    The QR Code will point to the current page

    Please insure that the page is public

Page

    The QR Code will point to one of your pages

    Please insure that the page is public

vCard

    Creates a QR containing a vCard with the information given by you

    Please bear in mind, that the either the First Name field or the Last name must be filled and that the QR Code created maybe become too complex if too much info is added

meCard

    Creates a QR containing a meCard with the information given by you

    Please bear in mind, that the either the First Name field or the Last name must be filled

Email Address

    The QR Code will point to a email address

Email Message

    The QR Code will contain a email address and message that you can send.

    The fields Email and Text are mandatory.

Phone number

    The QR Code will contain a Phone Number

SMS message

    The QR Code will contain a SMS message prefilled with the text and phone number chosen by you

Text

    The created QR Code will contain a text written by you

Website URL

    The QR Code will point to a website of your choosing
