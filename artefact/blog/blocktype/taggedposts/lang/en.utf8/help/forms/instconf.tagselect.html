Choosing tags

You can add multiple tags to limit the results as well as indicate which tags you want to ignore.

If you select multiple tags, a journal entry must match ALL selected tags in order to be listed.

If you had journal entries tagged like so:

  1. cats, domestic, animals
  2. cats, domestic, animals, small
  3. cats, domestic, animals, large
  4. cats, wild, animals
  5. cats, wild, animals, small
  6. cats, wild, animals, large
  7. dogs, wild, animals
  8. dogs, wild, animals, small
  9. dogs, wild, animals, large

and you:

  • select the tag "animals", all journal entries would be displayed in the block.
  • select the tags "cats" and "large", journal entries 3 and 6 would be displayed.
  • select the tags "animals" and "large", journal entries 3, 6, and 9 would be shown.

Excluding tags:

You can also exclude tags from the results by putting a minus sign before the tag you want to select.

If you:

  • select the tag "cats" and exclude "domestic" (exclude the tag by typing "-domestic" without the quotation marks), journal entries 4, 5, and 6 would be displayed.
  • select the tag "wild" and "animals" and exclude "small" and "large" (exclude the tag by typing "-small" "-large" without the quotation marks), journal entries 4 and 7 would be shown.
